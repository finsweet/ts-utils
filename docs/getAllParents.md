## `getAllParents()`

This util extracts a list of all parents of an element, excepting the `<body>` and `<html>` tags

| param              | value              |
| ------------------ | ------------------ |
| element: `Element` | The target element |

| Return value: `HTMLElement[]`               |
| ------------------------------------------- |
| The first child node text element, if found |
