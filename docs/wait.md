## `wait()`

This util returns an awaitable promise for waiting X time.

| param          | value      |
| -------------- | ---------- |
| time: `number` | Time in ms |

| Return value: `Promise<unknown>`      |
| ------------------------------------- |
| Awaitable promise for waiting X time. |
