## `removeTrailingSlash()`

Compares if two arrays have the same exact values.

| param           | value                |
| --------------- | -------------------- |
| value: `string` | The value to mutate. |

cc
| A new string without a trailing slash. |

```
 This:
 https://www.finsweet.com/attributes/attractions/capri-island/

 Becomes:
 https://www.finsweet.com/attributes/attractions/capri-island
```
