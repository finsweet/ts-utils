## `removeChildElements()`

This util removes all child elements from a parent element

| param               | value                                                       |
| ------------------- | ----------------------------------------------------------- |
| element: `Element`  | Parent element                                              |
| selector?: `string` | Optional: only remove the elements that match this selector |
